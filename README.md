# 小红书种草手写标注工具

一个本地可用的小工具，用来把餐食、饮品、店铺环境或商品图做成“小红书日常分享”风格的手写标注图。

## 怎么用

直接打开 `index.html`：

```bash
open index.html
```

也可以用浏览器打开这个文件。上传图片后，选择模板、改文案、拖动画布上的文字位置，最后点“导出 PNG”。

## 适合场景

- 商家上新菜单、饮品、甜品、套餐
- 探店/种草图
- 小红书封面或正文配图
- 店铺氛围图、窗边座位、下午茶套餐

## 当前功能

- 上传本地图片
- 预设模板：咖啡店餐食、饮品种草、甜品探店、店铺氛围
- 白色手绘线条、虚线箭头、简单轮廓
- 中文短句碎碎念
- 快捷词添加
- 导出 1080 x 1440 PNG

## 后续可以加

- 接 GPT Image 做更自然的手写风格
